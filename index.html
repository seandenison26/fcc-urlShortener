Timestamp Service!

Pass a string as a query parameter either as a natural date or a unix time code and you will get back both for the date entered.

Example usage:

  • https://timestamp-ms.herokuapp.com/december%2015,%202015
  • https://timestamp-ms.herokuapp.com/1450137600

Example usage:

  • { "unix": 1450137600, "natural": "December 15, 2015" }
